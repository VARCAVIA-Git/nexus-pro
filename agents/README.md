# Nexus Pro — OpenClaw Agents

This directory contains AI agent configurations for OpenClaw integration.

## Agents

- `market-analyst` — Analyzes market conditions and generates reports
- `signal-generator` — Processes technical signals and validates trades
- `risk-manager` — Monitors portfolio risk and enforces limits

## Setup

Ensure OpenClaw is installed at `~/.openclaw` and configured with your API keys.
